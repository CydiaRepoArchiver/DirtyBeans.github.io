° Installing this package will add the Cydia Auto Install folder.
° After install:
    1. Copy deb packages to:
        /private/var/root/Media/Cydia/AutoInstall/
    2. Rebooting device will install whatever packages are in that folder.
    Warning: Some debs need the depend package as well.
